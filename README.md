# Kino Traffic

Kino Traffic is a film recommendation web app.

A user searches for a film they like, confirms the selected film, chooses what they liked about it, and receives ranked recommendations based on creative overlap such as direction, acting, cinematography, and music.

## MVP Goal

The first version will be a small, understandable web app that proves the core user journey:

1. Search for a film
2. Select the correct film
3. Choose what the user liked about it
4. Highlight relevant cast and crew
5. Show ranked recommendations with match reasons

## First Build Scope

The first build will use fake data only.

It will include:

- Static homepage
- Search bar
- Fake autocomplete dropdown
- Selected film layout
- Category selection
- Fake cast and crew pills
- Fake recommendation grid

## Later Features

- TMDB movie search
- FastAPI backend
- Real recommendation scoring
- Tests
- GitHub Actions
- Docker
- Deployment
