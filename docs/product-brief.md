# Kino Traffic Product Brief

## Concept

Kino Traffic is a film recommendation app based on creative overlap.

Instead of only recommending films by genre or popularity, it lets the user choose what they liked about a film, such as direction, acting, cinematography, or music.

The app then recommends films that share relevant creative people, roles, or metadata.

## Core User Flow

1. User lands on the homepage,
2. User types the title of a film they like,
3. Search results appear in a dropdown,
4. User selects a film,
5. App shows the selected film page,
6. User confirms whether this is the correct film,
7. User chooses one or more categories,
8. Relevant cast and crew are highlighted,
9. User submits the selection,
10. App shows ranked film recommendations,
11. Each recommendation explains why it matched.

## MVP Principle

The first version should be simple and explainable.

The app should avoid machine learning at first. Recommendations should use clear rule-based scoring.
